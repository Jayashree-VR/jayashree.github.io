SmartDine AI

A sophisticated AI recommendation engine that translates subjective cravings into personalized culinary suggestions using Natural Language Processing.

Project Information

  • Category: Artificial Intelligence / Web Dev
  • Tech Stack: HTML, CSS, Javascript, Spring Boot, MySQL
  • Project Date: January 2026
  • Status: Completed

Key Deliverables

  • Engineered a high-performance recommendation engine using Spring Boot.
  • Developed a Natural Language Processing (NLP) module to handle complex user prompts.
  • Integrated Sentiment Analysis to refine dish suggestions based on user mood and cravings.
  • Designed a dual-pane dashboard featuring a live menu and an AI chatbot interface.
  • Implemented an intelligent mapping system to connect keyword patterns to menu item metadata.

Detailed Description

SmartDine redefines the digital menu experience by moving beyond static filters. At its core, the project utilizes a Spring Boot backend to power a sophisticated AI recommendation engine. Unlike traditional systems that rely on simple categories, SmartDine analyzes complex user descriptions—such as "something spicy and comforting for a rainy day"—and translates these subjective inputs into specific, data-driven menu recommendations.

The system features an intelligent mapping architecture that performs keyword extraction and sentiment analysis in real-time. This allows the application to understand the nuance behind "cravings," matching the user's emotional and physical preferences with the flavor profiles of the available dishes. The frontend provides a seamless split-screen dashboard where users can interact with an AI chatbot on one side and see their personalized menu update dynamically on the other.
