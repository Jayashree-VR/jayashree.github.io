Hospital Management System (Full-Stack MERN)

A comprehensive full-stack web application built with the MERN stack for managing hospital workflows efficiently.

Project Information

  • Category: Web Application / Full-Stack
  • Technologies/Tools: HTML, CSS, Javascript, MERN
  • Project Date: April 2025
  • Status: Ongoing

Key Deliverables

  • Built the entire frontend using HTML, CSS, and JavaScript.
  • Developed responsive pages for patients, doctors, appointments, and billing.
  • Planned the database structure for future MongoDB integration.
  • Currently learning the MERN stack to develop the backend and REST APIs.

Detailed Description

This Hospital Management System currently includes a fully developed frontend built using HTML, CSS, and JavaScript. The interface covers major hospital modules such as patient registration, doctor details and appointments. The layout is designed to be clean, intuitive, and responsive, making it easy for staff and administrators to navigate.

The project is structured with future integration in mind. I am in the process of learning the MERN stack (MongoDB, Express.js, React.js, Node.js) and will be building the backend and database next. The planned backend will include REST APIs for managing patients, appointments, billing, and authentication. MongoDB will be used to create flexible database schemas suitable for healthcare data.

Although the backend is not yet implemented, the frontend is fully prepared for integration. The UI is designed in a modular way so it can be converted to React components later. This project reflects my skills in frontend development and my ongoing progress toward becoming a full-stack MERN developer.
