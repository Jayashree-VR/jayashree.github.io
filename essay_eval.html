Automated Essay Evaluation System

A frontend-based writing assistance tool that allows users to generate essay topics, set custom word limits and timers, write essays, and receive instant AI-generated feedback on grammar, vocabulary, structure, and clarity.

Project Information

  • Category: Web Application (Frontend-only)
  • Technologies/Tools: HTML, CSS, JavaScript, AI API
  • Project Date: November 2025
  • Status:Ongoing

Key Deliverables

  • Built a fully responsive frontend interface for essay writing and evaluation.
  • Implemented genre selection with random topic generation for creative writing.
  • Added customizable word limits and an integrated writing timer.
  • Integrated AI for real-time evaluation of grammar, vocabulary, clarity, and structure.
  • Displayed instant feedback ratings to help users improve writing skills.

Detailed Description

The Automated Essay Evaluation System is a browser-based writing tool designed to improve users’ writing skills using AI-powered feedback. The interface allows users to select a genre, generate random essay topics, and begin writing with a self-set word limit and timer. Once the essay is submitted, the system sends the content to an AI model, which provides structured feedback on grammar, vocabulary richness, sentence clarity, coherence, and overall writing quality. This eliminates the need for backend servers, making the project lightweight and efficient. The system serves as a quick practice platform for students preparing for writing exams and helps improve writing skills with instant, actionable feedback.
