PhysioTrack

An IoT-enabled wearable device for physiotherapy, tracking joint movements and exercise repetitions in real-time. The system provides a web dashboard for patients and physiotherapists to monitor rehabilitation progress efficiently.

Project Information

  • Category: IoT + Embedded Systems
  • Components: Arduino, IMU Sensors, Web Dashboard
  • Project Date: October 2025
  • Status: Ongoing

Key Deliverables

  • Developed a wearable IoT device to track joint angles and exercise repetitions.
  • Integrated Arduino with IMU sensors for accurate motion detection.
  • Created a responsive web dashboard to visualize real-time patient data.
  • Implemented secure data transmission from wearable device to the cloud.
  • Demonstrated improved physiotherapy monitoring through automation and analytics.

Detailed Description

PhysioTrack is a wearable physiotherapy monitoring system that captures joint movements and exercise repetitions using IMU sensors connected to an Arduino microcontroller. The device transmits real-time data to a web-based dashboard, enabling patients and physiotherapists to track rehabilitation progress efficiently. The system combines seamless hardware-software integration with battery-powered mobility, accurate motion tracking, and intuitive visualization.

The patient dashboard provides real-time visualization of joint angles and repetitions, tracks daily and historical progress, and delivers exercise instructions. To encourage consistent practice, patients earn streaks for consecutive exercise days and badges for achieving milestones, motivating continued engagement. The doctor dashboard allows physiotherapists to monitor multiple patients, analyze performance trends, set custom goals, and provide targeted feedback, facilitating remote and data-driven rehabilitation management.
